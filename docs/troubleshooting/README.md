# Troubleshooting Guides

This directory contains guides to help you resolve common issues with the Omni Care Software system.

## Available Guides

### Database Issues

- **[Missing Database Columns](MISSING_DATABASE_COLUMNS.md)** - Resolves errors like `column "IsPaid" of relation "Appointments" does not exist` and similar database schema mismatch issues.

## How to Use These Guides

1. Find the guide that matches your issue
2. Follow the recommended solution (usually Option 1)
3. If the issue persists, try the alternative solutions in order
4. Check the "Getting Help" section if you still need assistance

## Contributing

If you encounter an issue not covered in these guides, please:

1. Document the issue and its solution
2. Create a new troubleshooting guide following the existing format
3. Add it to this index
4. Submit a pull request
