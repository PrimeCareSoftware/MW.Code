## Política de Arquivamento

Arquivamos documentação quando:
1. **Consolidação**: Múltiplos documentos similares foram mesclados em um único documento abrangente
2. **Supersedência**: Uma nova versão mais completa substitui versões anteriores
3. **Redundância**: Informações duplicadas existem em documentos mais atualizados
4. **Histórico**: Status temporários que já não refletem o estado atual

Todos os arquivos arquivados são mantidos para referência histórica.
